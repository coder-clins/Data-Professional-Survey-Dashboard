# Data Professional Dashboard - An Insight into the Professionals Working in Data
A dynamic, interactive data visualization tool built to find key insights about the professionals working in data domain

# 2. Short Description / Purpose
Data Professional Dashboard is a visually engaging and analytical PowerBI report designed to find key insights about professionals working in the field of Data across different countries and continents. The dashboard focuses on finding the most popular programming language among these professionals, average salary that a data professional can expect to earn in various data domain jobs in various countries, how satisfied the professionals are with their jobs and their salaries, and the difficulty level they found to break into data jobs. This repoert is intended for use by current data professionals as well as students and professionals looking for a job in Data domain fields who want to understand the nature of these jobs.

# 3. Tech Stack
The dashboard was built using the following tools and technologies:
. Power BI Desktop - Main data visualization platform used for reoirt creation.
. Power Query - Data transformation and cleaning layer fir reshaping and preparing the data
. DAX (Data Analysis Expressions) - Used fir calculated measures dynamic visuals, and conditional logic.
. Data Modeling - Relationship established among tables (resorts, snow, and data_dictionary) to enablie cross-filtering and aggregation.
